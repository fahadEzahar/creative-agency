# Creative Agency - Full stack website for a software agency with responsive UI
[Live Website](https://creative-agency-72c79.web.app/)
##### Overview: 
* Customer can order service, check order status and write review.
* Admin can add services, make admin, and change order status.
##### Technology used:
* JaveScript (ES6)
* React
* NodeJS
* MongoDB
* Bootstrap
* CSS3
* HTML5
* Firebase
* Heroku
